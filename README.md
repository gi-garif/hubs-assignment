### Assignment description:
Write an automated test to check model uploading on https://www.hubs.com/manufacture/using Python 3.8 (or any latest version) and PyTest

### Requirements:
* Python 3.10.6
* Selenium 4.6.0
* webdriver-manager 3.8.5
* Pytest 7.2.0
* Google Chrome
